### Project Description
[Natural disasters like floods and earthquakes create confusion because people often lack accurate and timely information about safe shelters and emergency help.
During events such as the 2018 Kerala floods, many people struggled to find real-time updates about relief camps and support services.
To solve this problem, our website “RescueLink” provides live safe zone maps, relief camp details, emergency contacts, and a missing person reporting system to ensure quick access to life-saving information.]
